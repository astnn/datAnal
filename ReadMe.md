This repo host Python-code solving the exercises of the DTU (www.dtu.dk) course
"30230 - Data Analysis and Modeling in Geoscience and Astrophysics" by Nils
Olsen. See https://kurser.dtu.dk/course/30230 for the course description.
The presented code is based on the lectures and exercises as given in 2018.
Matlab code solving the exercises is distributed as part of the course.
